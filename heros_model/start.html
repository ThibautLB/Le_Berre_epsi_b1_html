Le début d'un mythe

Après un long voyage, vous arrivez dans une clairière où se dresse un étrange monument. Celui-ci semble être surmonté d'une magnifique épée scellée dans la roche.

  • 1) Je m'approche prudemment de l'épée.
  • 2) Ça sent le piège, je passe mon chemin.
  • 3) Je sors mon arc pour parer à tout danger.
